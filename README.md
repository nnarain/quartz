# Quartz

[![Build Status](https://travis-ci.org/nnarain/quartz.svg?branch=master)](https://travis-ci.org/nnarain/quartz)

Quartz is a Chip8 virtual machine written in the Rust programming language.

Usage
-----

To run a rom file:

```
quartz /path/to/romfile
```

To dipslay options:

```
quartz -h
```

Build
-----

```bash
cargo build --release
```
